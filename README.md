# Falashi
A ideia surgiu de uma percepção simples dentro do grupo: estudar com flashcards funciona, mas é chato. Ferramentas de estudo existentes são tecnicamente eficientes, mas não têm nenhum elemento que motive o usuário a continuar além da força de vontade. O Falashi é um app de estudo gamificado onde o usuário responde perguntas de flashcards falando no microfone do dispositivo esse é o nosso principal diferencial em relação a tudo que já existe no mercado. Em vez de digitar ou apertar botões, o jogador fala a resposta em voz alta, tornando a experiência mais dinâmica e natural. O app combina modos solo e multiplayer, sistema de ranking, moeda virtual e personalização, transformando o estudo em uma atividade competitiva e envolvente. A proposta não é reinventar o flashcard, é mudar a relação do estudante com o ato de estudar.
